FAQ

Does it work on every store?

It works on any store running Shopify checkout. That's most independent e-commerce stores. It won't work on Amazon, eBay, or custom-built checkouts.

Is it free?

Yes. Completely free, open source, no ads, no tracking. Install from GitHub.

Will it get me banned?

Shopify may show a "too many attempts" warning if you test a lot of codes quickly. It's cosmetic and clears after a few minutes. Your order is fine.

How does it find codes?

Three ways. It searches Reddit threads for people sharing codes, scrapes coupon aggregator sites, and generates likely codes based on the brand name and current holidays.

Does it send my data anywhere?

No. Everything runs locally in your browser. The only network requests go to Reddit and coupon sites to find codes. Nothing is sent to us because there is no "us".
